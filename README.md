# basic

basic基础库

## 安装

使用 ohpm 安装：bash
ohpm install hmos_basic
或
bash
ohpm i hmos_basic
## 使用说明

### 基本使用

basic基础库提供了一系列基础工具函数和工具类，帮助开发者更高效地进行HarmonyOS应用开发。

### 示例代码
typescript
// 导入基础库
import { 工具函数 } from 'hmos_basic';
// 使用示例
// 这里添加具体的使用示例代码
### 主要功能

- **工具函数**: 提供常用的工具方法
- **工具类**: 封装常用的功能类
- **工具模块**: 模块化的工具集合
